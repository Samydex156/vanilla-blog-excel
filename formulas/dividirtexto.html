Función DIVIDIRTEXTO()

Divide cadenas de texto mediante delimitadores y las desborda automáticamente en múltiples columnas o filas. Es la versión en fórmula de la clásica herramienta de datos "Texto en Columnas".

Sintaxis

=DIVIDIRTEXTO(texto; col_delimitador; [row_delimitador]; [ignorar_vacío]; [modo_coincidencia]; [pad_with])
texto: La cadena de texto o celda que contiene el texto que deseas separar (obligatorio).
col_delimitador: El carácter o caracteres de separación que indican dónde dividir el texto en columnas horizontales (obligatorio).
row_delimitador: El carácter o caracteres que indican dónde dividir el texto en filas verticales (opcional).
ignorar_vacío: Valor lógico. Usa VERDADERO para ignorar delimitadores consecutivos seguidos (opcional, por defecto es FALSO).
pad_with: El valor que se usará para rellenar celdas vacías en caso de generar matrices irregulares de filas y columnas (opcional, por defecto muestra error #N/A).

Ejemplos Prácticos

  • Dividir en columnas (horizontal): Si la celda A1 tiene el texto "Santiago, Valparaíso, Concepción":
    =DIVIDIRTEXTO(A1; ", ") ➔ Devolverá tres celdas horizontales continuas con cada una de las ciudades.
  • Dividir en filas (vertical): Para lograr que las mismas ciudades queden ordenadas hacia abajo en la misma columna, se usa el tercer argumento:
    =DIVIDIRTEXTO(A1; ; ", ") *(dejamos el segundo argumento vacío).*

Observaciones

  • Error de Desbordamiento: Al ser una función matricial dinámica, si las celdas contiguas donde se desbordará la información están ocupadas, devolverá el error #¡DESBORDAMIENTO!.
  • En la versión en inglés de Excel, esta función se denomina TEXTSPLIT().
  • *Nota: Esta función es muy moderna y está disponible exclusivamente en Excel para Microsoft 365 y Excel Web.*
