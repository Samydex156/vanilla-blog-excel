Función SI.ERROR()

Evalúa una fórmula o expresión y, si detecta un error de Excel, devuelve un valor alternativo personalizado que definas. Evita que tus hojas muestren códigos extraños como #DIV/0! o #N/A.

Sintaxis

=SI.ERROR(valor; valor_si_error)
valor: La fórmula, cálculo, función o referencia de celda que deseas evaluar para comprobar si contiene un error (obligatorio).
valor_si_error: El texto, número o cálculo alternativo que deseas mostrar si se detecta un error en el primer argumento (obligatorio).

Ejemplo Práctico

Supongamos que deseas dividir el total de ventas (en A1) entre la cantidad de vendedores (en B1). Si B1 está vacío o contiene 0, Excel arrojará el error #DIV/0!. Para solucionarlo:

=SI.ERROR(A1/B1; 0)

Si la división es normal, mostrará el resultado real. Si hay un error, mostrará un limpio **0**.

Otro caso común es envolver búsquedas de códigos:

=SI.ERROR(BUSCARV(D1; A2:B10; 2; FALSO); "Código inexistente")

Observaciones

  • La función intercepta y maneja todos los errores estándar de Excel: #N/A, #¡VALOR!, #¡REF!, #¡DIV/0!, #¡NUM!, #¿NOMBRE? y #¡NULO!.
  • Si el argumento valor no genera ningún error, la función devuelve el resultado normal del cálculo sin aplicar ninguna modificación.
  • En la versión en inglés de Excel, esta función se denomina IFERROR().
