Función ORDENAR()

Ordena los valores de un rango o matriz de datos de forma automática, dinámica y en tiempo real. Al pertenecer a la nueva generación de funciones matriciales, el resultado se desborda en múltiples celdas contiguas.

Sintaxis

=ORDENAR(matriz; [ordenar_índice]; [criterio_ordenación]; [por_columna])
matriz: El rango de celdas o la tabla que deseas ordenar (obligatorio).
ordenar_índice: El número de fila o columna por la cual ordenar (ej. 2 para ordenar por la segunda columna) (opcional, por defecto es 1).
criterio_ordenación: Dirección del orden: 1 para orden ascendente (A-Z, menor a mayor) o -1 para orden descendente (Z-A, mayor a menor) (opcional, por defecto es 1).
por_columna: Un valor lógico. Usa FALSO para ordenar filas (lo habitual) o VERDADERO para ordenar horizontalmente por columnas (opcional, por defecto es FALSO).

Ejemplo Práctico

Supongamos que tienes una tabla de nombres y puntajes en A2:B10, donde la columna B tiene los puntajes. Deseas ordenar la lista de mayor a menor puntaje de forma dinámica:

=ORDENAR(A2:B10; 2; -1)

Excel tomará toda la tabla y desbordará el resultado ordenado a partir de la celda donde insertes la fórmula. Si agregas o cambias puntajes en la tabla original, el resultado se actualizará al instante.

Observaciones

  • Error de Desbordamiento: Si las celdas a las cuales se debe desbordar la lista ordenada contienen otros datos (incluso un solo espacio en blanco), Excel devolverá el error #¡DESBORDAMIENTO! (o #SPILL!). Limpia esas celdas para resolverlo.
  • En la versión en inglés de Excel, esta función se denomina SORT().
  • *Nota: Esta función solo está disponible en Excel para Microsoft 365, Excel 2021 y Excel Web.*
