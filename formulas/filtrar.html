Función FILTRAR()

Extrae y muestra de forma dinámica todas las filas o registros de una tabla que cumplan con una o más condiciones especificadas. Pertenece a las nuevas funciones de matrices dinámicas.

Sintaxis

=FILTRAR(matriz; incluir; [si_vacío])
matriz: El rango completo de datos o columnas de la tabla que deseas filtrar (obligatorio).
incluir: Una comparación lógica (o conjunto de comparaciones) que define los criterios de filtrado. Sus dimensiones deben coincidir con las de la matriz (ej. B2:B10="Lácteos") (obligatorio).
si_vacío: El texto, número o mensaje que deseas mostrar en caso de que ningún registro de la tabla cumpla con la condición dada (opcional, por defecto devuelve error #CALC!).

Ejemplos Prácticos

Supongamos una tabla de ventas en A2:C10 donde la columna B contiene la categoría del producto:

  • Para filtrar todas las ventas de la categoría "Juguetes":
    =FILTRAR(A2:C10; B2:B10="Juguetes"; "No hay ventas")
  • Para filtrar con múltiples condiciones (ej. categoría "Juguetes" **Y** ventas en columna C mayores a 100):
    =FILTRAR(A2:C10; (B2:B10="Juguetes") * (C2:C10>100); "Sin resultados") *(se usa el signo asterisco * para representar la condición lógica Y)*
  • Para filtrar con condición O (ej. categoría "Juguetes" **O** categoría "Libros"):
    =FILTRAR(A2:C10; (B2:B10="Juguetes") + (B2:B10="Libros"); "Sin resultados") *(se usa el signo de suma + para representar la condición lógica O)*

Observaciones

  • Error de Desbordamiento: Al igual que ORDENAR(), requiere que el área de salida del filtrado esté completamente despejada para evitar el error #¡DESBORDAMIENTO!.
  • En la versión en inglés de Excel, esta función se denomina FILTER().
  • *Nota: Esta función solo está disponible en Excel para Microsoft 365, Excel 2021 y Excel Web.*
