Función LET()

Asigna nombres descriptivos a los resultados de cálculos u operaciones intermedias dentro de una misma fórmula. Esto permite almacenar variables, evitando repetir operaciones redundantes, lo que optimiza la velocidad y legibilidad.

Sintaxis

=LET(nombre1; valor_nombre1; [nombre2; valor_nombre2]; ...; cálculo_final)
nombre1: El nombre que deseas dar a la primera variable o resultado intermedio. Debe comenzar por una letra y no contener espacios (obligatorio).
valor_nombre1: El valor numérico, texto o fórmula de cálculo que se asignará al nombre1 (obligatorio).
nombre2 / valor_nombre2; ...: Parejas adicionales opcionales de nombres y valores asignados (opcional, hasta 126 variables).
cálculo_final: El cálculo u operación matemática definitiva que utilizará las variables declaradas previamente para devolver el resultado. Debe ser el último argumento de la función (obligatorio).

Ejemplos Prácticos

  • Ejemplo Básico de Suma:
    =LET(x; 10; y; 20; x + y) ➔ Devolverá 30.
  • Ejemplo Real de Optimización: Si deseas calcular un bono del 10% si la suma de ventas de un vendedor supera los 10,000, o de lo contrario un 5%:
    =LET(ventas; SUMA(C2:C30); SI(ventas > 10000; ventas * 0.1; ventas * 0.05))
    *(En una fórmula normal tendrías que escribir SUMA(C2:C30) tres veces. Con LET(), Excel calcula la suma una sola vez en memoria, acelerando el archivo radicalmente).*

Observaciones

  • Mejora del rendimiento: Si declaras fórmulas complejas en las variables (como búsquedas BUSCARV()), Excel las ejecuta una sola vez y almacena el resultado en memoria, reduciendo el consumo de CPU en hojas con miles de registros.
  • Los nombres de las variables no pueden coincidir con los nombres de las funciones existentes de Excel (por ejemplo, no puedes crear una variable llamada SUMA).
  • *Nota: Disponible exclusivamente en Excel para Microsoft 365, Excel 2021 y Excel Web.*
