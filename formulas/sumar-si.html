Función SUMAR.SI()

Suma los valores de un rango que cumplen con una condición o criterio determinado. Ideal para reportes financieros y consolidados de datos.

Sintaxis

=SUMAR.SI(rango; criterio; [rango_suma])
rango: El rango de celdas que deseas evaluar con la condición o criterio (obligatorio).
criterio: La condición que determina qué celdas se van a sumar. Puede ser un número (ej. 100), una expresión (ej. ">50"), un texto (ej. "Frutas") o una referencia a una celda (obligatorio).
rango_suma: Las celdas reales que se van a sumar. Si se omite, Excel sumará las celdas especificadas en el primer argumento (rango) (opcional).

Ejemplos Prácticos

Supongamos una tabla de ventas donde la columna B contiene los nombres de los vendedores y la columna C el monto vendido:

  • Para sumar solo las ventas realizadas por "Juan":
    =SUMAR.SI(B2:B10; "Juan"; C2:C10)
  • Para sumar directamente todos los montos de la columna C mayores a 500:
    =SUMAR.SI(C2:C10; ">500") *(aquí se omite el argumento rango_suma porque se evalúa y suma el mismo rango)*

Observaciones

  • Al igual que en CONTAR.SI(), cualquier criterio lógico que contenga operadores de comparación debe ir escrito entre comillas dobles (ej. "<=100").
  • La función soporta el uso de caracteres comodín (* y ?) para criterios de texto parciales. Por ejemplo, =SUMAR.SI(B2:B10; "*Norte*"; C2:C10) sumará las ventas de zonas que contengan la palabra "Norte" (como "Norte Centro", "Norte Chico").
