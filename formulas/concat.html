Función CONCAT()

Une múltiples cadenas de texto o rangos de celdas en una sola celda. Es la versión mejorada y recomendada que reemplaza a la antigua función CONCATENAR.

Sintaxis

=CONCAT(texto1; [texto2]; ...)
texto1: El primer elemento de texto, celda o rango de celdas completo que deseas unir (obligatorio).
texto2; ...: Elementos de texto, celdas o rangos adicionales (hasta 254) que deseas unir (opcional).

Ejemplos Prácticos

  • Unir celdas con espacios intermedios: Si A2 contiene "Samuel" y B2 contiene "Durán":
    =CONCAT(A2; " "; B2) ➔ Devolverá "Samuel Durán".
  • Unir rangos completos: Si las celdas A1:A3 contienen "A", "B" y "C":
    =CONCAT(A1:A3) ➔ Devolverá "ABC". *(Esta acción no se podía realizar con CONCATENAR, tenías que seleccionar celda por celda).*

Observaciones

  • Diferencia clave con CONCATENAR: La función clásica CONCATENAR() no permitía rangos (ej. A1:A3). CONCAT() acepta tanto celdas individuales como rangos de forma directa.
  • Si el resultado final supera el límite de caracteres permitido para una celda de Excel (32,767 caracteres), la función devolverá el error #¡VALOR!.
  • Si requieres unir un rango separando cada elemento por un carácter (como comas, guiones o espacios) e ignorando celdas vacías, se recomienda utilizar la función UNIRCADENAS().
