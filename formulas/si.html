Función SI()

Evalúa una condición y devuelve un resultado si se cumple (VERDADERO) y otro si no se cumple (FALSO). Es la base del análisis lógico en Excel.

Sintaxis

=SI(prueba_lógica; valor_si_verdadero; [valor_si_falso])
prueba_lógica: La condición lógica que deseas evaluar. Puede ser una comparación como A1>=18, B2="Aprobado", etc. (obligatorio).
valor_si_verdadero: El resultado que devolverá la función si la condición se cumple (obligatorio).
valor_si_falso: El resultado que devolverá la función si la condición no se cumple (opcional, por defecto devuelve el texto "FALSO").

Ejemplo Práctico

Imagina que tienes la nota de un alumno en la celda A1 y quieres saber si está aprobado o reprobado (la nota de aprobación es 60):

=SI(A1>=60; "Aprobado"; "Reprobado")

Si A1 tiene una nota de 75, la fórmula devolverá "Aprobado". Si tiene 45, devolverá "Reprobado".

Observaciones

  • Es posible anidar múltiples funciones SI() para evaluar más de dos opciones. Por ejemplo: =SI(A1>=90; "Sobresaliente"; SI(A1>=60; "Aprobado"; "Reprobado")).
  • Para condiciones de texto en el argumento prueba_lógica, recuerda usar siempre comillas dobles (ej. A1="Si"). Excel no distingue entre mayúsculas y minúsculas al comparar texto en esta función.
