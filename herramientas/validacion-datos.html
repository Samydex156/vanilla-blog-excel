Validación de Datos

Garantiza la integridad de tu información restringiendo el tipo de datos permitidos.

¿Por qué usar validación?

La validación de datos ayuda a evitar errores de entrada (como escribir un texto donde debería ir un número) y permite crear menús desplegables fáciles de usar.

Tipos de Restricciones

Lista: Crea un menú desplegable con opciones predefinidas.
Número Entero: Solo permite valores numéricos sin decimales.
Fecha: Restringe la entrada a un rango de fechas específico.

Cómo configurarla

Ve a la pestaña Datos > Validación de datos. En la pestaña "Configuración", elige el criterio que necesites. También puedes personalizar el mensaje de error que verá el usuario si ingresa un dato incorrecto.

Caso de uso común: Crear un desplegable para seleccionar "SÍ" o "NO" en una columna de estado.
