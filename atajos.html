Lista de Atajos de Teclado

Ahorra horas de trabajo eliminando el uso innecesario del ratón. Aquí tienes todos los shortcuts unificados para el curso.

Fundamentales (Edición y Archivo)

Ctrl + G Guardar / Actualizar cambios en el libro actual. F12 Guardar por primera vez / Guardar una copia del archivo. Ctrl + P Imprimir el documento / Ver vista previa de impresión. Ctrl + C Copiar selección. Ctrl + V Pegar selección. Ctrl + X Cortar selección. Ctrl + Z Deshacer última acción. Ctrl + Y Rehacer última acción. Ctrl + J Rellenar hacia abajo (Copia la celda superior). Ctrl + ' Clonar el valor de la celda de arriba. Ctrl + B Buscar un valor en la hoja. Ctrl + L Reemplazar un valor por otro.

Selección y Navegación

Ctrl + Flechas Saltar al límite de los datos / Ir al final de la hoja Ctrl + Shift + Flechas Seleccionar un rango de datos completo. Ctrl + E Seleccionar toda la tabla actual. Ctrl + * Seleccionar toda la tabla actual. Shift + Espacio Seleccionar fila completa. Ctrl + Espacio Seleccionar columna completa. Ctrl + Inicio Ir a la celda A1. Ctrl + Retroceso Volver visualmente a la celda activa. Ctrl + AvPág / RePág Cambiar entre pestañas (hojas) del libro. Shift + F11 Insertar una nueva hoja de cálculo en el libro.
Consejo: Combina Ctrl + Shift para realizar selecciones ultrarrápidas de tablas con miles de registros.

Formato y Datos

Ctrl + 1 Abrir ventana de Formato de Celdas. Ctrl + N Poner texto en Negrita. Ctrl + K Poner texto en Cursiva. Ctrl + T Convertir rango en Tabla oficial. Alt + Shift + 0 AutoSuma automática de valores. Ctrl + Shift + L Activar / Desactivar Autofiltros. Ctrl + , Insertar la Fecha Actual (Valor fijo). Ctrl + Shift + . Insertar la Hora Actual (Valor fijo).

Poder y Funciones

F2 Editar la celda seleccionada (sin borrar el contenido). F4 Repetir última acción O Fijar referencias (poner los $). Ctrl + Shift + Enter Ingresar fórmulas matriciales (versiones antiguas). Alt + F1 Crear un gráfico instantáneo en la misma hoja. F11 Crear un gráfico automático en una hoja nueva independiente.
