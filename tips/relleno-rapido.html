Relleno Rápido (Flash Fill)

La forma más inteligente de limpiar datos sin usar una sola fórmula.

¿Cómo funciona?

El Relleno Rápido detecta automáticamente patrones en tus datos. Si escribes un ejemplo de cómo quieres que se vean los datos en la celda adyacente, Excel intentará completar el resto por ti.

Pasos para usarlo

  1. Escribe el primer ejemplo en la celda de al lado de tus datos.
  2. Presiona Enter.
  3. En la siguiente celda, comienza a escribir el segundo ejemplo y Excel te mostrará una vista previa de cómo quedaría la tabla y pulsa Enter para completar el relleno. O También puedes presionar Ctrl + Shift + E. para rellenar.

¡Truco Maestro!

Puedes usarlo para separar nombres de apellidos, extraer números de un texto o incluso cambiar el formato de un número de teléfono en segundos.
