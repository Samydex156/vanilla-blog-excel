Duplicar Información al Instante

Copia el contenido de la celda superior sin necesidad de arrastrar o usar Copiar/Pegar.

Atajos de Clonación

Estos atajos son ideales cuando estás llenando una lista y el dato de arriba se repite. Funcionan como un "sello" que trae la información hacia abajo.

Ctrl + '
Clonar Valor: Copia exactamente lo que ves en la celda de arriba (el resultado) pero como un valor fijo.
Ctrl + J
Rellenar hacia abajo: Copia tanto el valor como el formato (y las fórmulas) de la celda superior.

¿Cuándo usar cada uno?

Usa Ctrl + ' cuando solo quieras el texto o número de arriba sin preocuparte por las fórmulas. Usa Ctrl + J cuando necesites que la fórmula de arriba se replique hacia abajo de forma masiva (puedes seleccionar varias celdas y presionar el atajo para rellenar todo el rango).

El "Sello" de Pegado de Valores

Si tienes una fórmula y quieres convertir su resultado en un valor fijo para que no cambie más, el atajo es:

  1. Copia la celda (Ctrl + C).
  2. Presiona Ctrl + Alt + V, luego presiona la letra V y Enter.

Esto "seca" la fórmula y deja solo el resultado final.
