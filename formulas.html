Biblioteca de Fórmulas

Consulta la sintaxis, argumentos y casos de uso de las funciones más potentes de Excel.

Nivel Básico

SUMA()

La función básica para sumar todos los números en un rango de celdas.

Matemáticas

PROMEDIO()

Calcula la media aritmética de un grupo de valores numéricos.

Estadística

MAX() y MIN()

Identifica el valor más alto y el más bajo dentro de una selección de datos.

Estadística

CONTAR()

Cuenta el número de celdas que contienen valores numéricos en un rango.

Estadística

CONTARA()

Cuenta la cantidad de celdas que no están vacías (números, texto, errores, etc.).

Estadística

CONTAR.BLANCO()

Cuenta el número de celdas vacías dentro de un rango específico.

Estadística

HOY()

Devuelve la fecha actual del sistema. Ideal para cálculos de edad y plazos.

Fecha y Hora

AHORA()

Devuelve la fecha y hora exactas. Útil para registros de tiempo precisos.

Fecha y Hora

DIA()

Extrae el día (número del 1 al 31) de una fecha específica.

Fecha y Hora

MES()

Devuelve el número de mes (del 1 al 12) correspondiente a una fecha.

Fecha y Hora

AÑO()

Extrae el año (número de cuatro dígitos) de una fecha determinada.

Fecha y Hora

MAYUSC()

Convierte todas las letras de una cadena de texto en mayúsculas.

Texto

MINUSC()

Convierte todas las letras de una cadena de texto en minúsculas.

Texto

NOMPROPIO()

Cambia a mayúscula la primera letra de cada palabra de un texto.

Texto

CONCAT()

Une una lista o rango de cadenas de texto de forma sencilla.

Texto

Nivel Intermedio

SIFECHA()

Calcula la diferencia en días, meses o años entre dos fechas dadas.

Fecha y Hora

SI()

Realiza una prueba lógica y devuelve un valor según el resultado sea VERDADERO o FALSO.

Lógica

SI.ERROR()

Captura y controla errores en fórmulas devolviendo un valor alternativo en caso de fallo.

Lógica

CAMBIAR()

Evalúa una expresión con una lista de valores y devuelve el primer resultado coincidente.

Lógica

CONTAR.SI()

Cuenta las celdas dentro de un rango que cumplen con un criterio especificado.

Estadística

SUMAR.SI()

Suma los valores de un rango que cumplen con una condición o criterio determinado.

Matemáticas

ESPACIOS()

Elimina los espacios iniciales, finales y dobles de una cadena de texto.

Texto

IZQUIERDA()

Devuelve el número especificado de caracteres desde el inicio de un texto.

Texto

DERECHA()

Devuelve el número especificado de caracteres desde el final de un texto.

Texto

EXTRAE()

Extrae caracteres de una cadena de texto, partiendo de una posición y longitud dadas.

Texto

TEXTO()

Convierte un valor numérico en texto y le aplica un formato específico.

Texto

UNIRCADENAS()

Une textos de varios rangos usando un delimitador y permitiendo ignorar celdas vacías.

Texto

BUSCARV()

Busca un valor en la primera columna de una tabla y devuelve el valor de otra columna en la misma fila.

Búsqueda

Nivel Avanzado

BUSCARX()

Busca un valor en un rango o matriz y devuelve el elemento correspondiente de un segundo rango o matriz.

Búsqueda

ORDENAR()

Ordena de forma automática y dinámica el contenido de un rango o matriz.

Búsqueda

FILTRAR()

Filtra un rango de datos basándose en los criterios lógicos que definas.

Búsqueda

DIVIDIRTEXTO()

Divide una cadena de texto en varias filas o columnas usando delimitadores específicos.

Texto

LET()

Asigna nombres a los resultados de cálculos intermedios para mejorar legibilidad y rendimiento.

Lógica

INDIRECTO()

Devuelve una referencia de celda o rango especificada mediante una cadena de texto.
