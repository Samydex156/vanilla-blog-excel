Atajos Fundamentales

Estos son los comandos básicos que acelerarán tu trabajo desde el primer día.

Gestión de Archivos y Edición

Ctrl + G Guardar el libro actual. Ctrl + C Copiar selección. Ctrl + V Pegar selección. Ctrl + X Cortar selección. Ctrl + Z Deshacer última acción. Ctrl + Y Rehacer última acción.

Formato Rápido

Ctrl + N Aplicar negrita al texto. Ctrl + K Aplicar cursiva. Ctrl + S Subrayar texto. Ctrl + 1 Abrir el cuadro de diálogo Formato de celdas.
