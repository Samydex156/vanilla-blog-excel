Selección y Navegación

Aprende a moverte por grandes volúmenes de datos sin despegar las manos del teclado.

Navegación Veloz

Ctrl + Flechas Salta hasta la última celda con datos en esa dirección. Ctrl + Inicio Vuelve a la celda A1 instantáneamente. Ctrl + Fin Va a la última celda utilizada en la hoja. AvPág / RePág Desplazarse una pantalla abajo o arriba.

Selección de Rangos

Shift + Flechas Extiende la selección celda por celda. Ctrl + Shift + Flechas Selecciona todo el rango de datos hasta el final. Ctrl + E Selecciona toda la región actual de datos. Shift + Espacio Selecciona la fila completa. Ctrl + Espacio Selecciona la columna completa.
Pro Tip: Combina Ctrl para moverte y Shift para seleccionar al mismo tiempo. Es la forma más rápida de marcar tablas grandes.
